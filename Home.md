# Awesome Heroes Wiki

To create a new page:
```markdown
[[new page link name|name of the file for new page]]
```
To make a header use the following:
```markdown
# Header 1 tag
## Header 2 tag
### Header 3 tag
#### Header 4 tag
##### Header 5 tag
###### Header 6 tag
```
# Header 1
## Header 2
### Header 3
